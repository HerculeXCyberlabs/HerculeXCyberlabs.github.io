✈️ AIRGAP / OFFLINE PC SETUP
Use the extension on air-gapped machines with no internet access:
01
On an online PC: Install extension → go to Feed tab → click "⬇ Sample Feed" to see format
02
Download IOC feeds from CERT-In, NCIIPC, MISP, or your SIEM exports as CSV/JSON/TXT
03
Import feeds in Feed tab → data stored in browser IndexedDB (survives restarts)
04
Copy extension folder to USB → install on airgap PC via chrome://extensions → Load unpacked
05
Import feeds on airgap PC from your USB IOC files → fully functional offline
INSTALLATION
Install in 4 Steps
Works on Chrome, Edge, Brave. Developer mode required (this extension is not on Chrome Web Store — it's free and open, installed directly from this page).
01
Download ZIP
Click the download button below to get the extension ZIP file.
02

03
Enable Dev Mode
Open Chrome → go to
chrome://extensions

04

Click Load unpacked → select the extension/ folder → done!
